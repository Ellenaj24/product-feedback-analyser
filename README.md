# Product Feedback Analyser

> A repeatable PM workflow for turning raw customer feedback into prioritised product opportunities, using Qualtrics, Excel and Microsoft Copilot.

## What this is

A personal project where I designed a complete workflow that takes customer feedback from multiple channels, consolidates it into one place, uses Copilot to surface themes and pain points, and produces a prioritised list of opportunities backed by real customer evidence.
